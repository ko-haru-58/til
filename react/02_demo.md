# デモ作成の記録
mdnでひと通りReactについて学習したあとに、デモを作成しています。  
金融関連で、期間や利子を設定し、積立を継続するとどれほど資産が増えるのかが分かる積立シミュレーションを作成中です。

## 2026/04/07(火)
入力欄の作成にあたり、お手本では入力欄のあるApp.jsxではなくmain.jsxでidなどのデータを渡している理由を調べました。

小規模なものではどちらでも構わないものの、実際のアプリはAPIなど外部からデータを取得することが多く、このような形になっているとのことでした。

## 2026/04/10(金)

ちょっとしたレベルの削除ミスで、ブラウザにすごい色でエラーが表示されてびっくりしました。Reactってこんな感じでエラーが出るんだなと思いました。
